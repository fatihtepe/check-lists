# Hello world-app
```
- copy ssh connection route from ec2 
- run the terminal  
- connect to EC2
- sudo yum update -y 
- sudo yum install python3 -y 
- python3 --version 
- sudo pip3 install flask 
- pip3 list 
- sudo yum install git wget -y 
- wget https://raw.github.com/... # copy raw url from github.com run it with wget command 
- sudo python3 hello-world.py 
- copy ec2 DNS address and run on explorer 
```
